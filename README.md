# X402 example server with Mogami

The Mogami x402 Spring Boot Starter adds a micropayment guard to any Spring Boot endpoint.
Full documentation is available at [here](https://mogami.gitbook.io/mogami/spring-boot-starter/example-server). 

Run the example server with the following command:

```bash
mvn spring-boot:run
```

You can make a test with the following command:

```bash
curl localhost:4021/weather
```
