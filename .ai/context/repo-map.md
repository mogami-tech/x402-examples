# Repo map

## mogami-tech/x402-commons
Shared foundation for building x402-compatible systems.

## mogami-tech/x402-java-client
Java client for x402, allowing users to easily integrate with the x402 protocol.
Depends on mogami-tech/x402-commons

## mogami-tech/x402-spring-boot-starter
Allows a user to protect its APIs with pay-per-call logic using a single Spring Boot annotation.
Depends on mogami-tech/x402-commons

## mogami-tech/x402-facilitator
X402 facilitator to verifies and settles payments.
Depends on mogami-tech/x402-commons

## mogami-tech/x402-examples
Java real-world demos of pay-walled endpoints and instant payment flows.
Use all the above libraries to demonstrate how to use them in practice.
